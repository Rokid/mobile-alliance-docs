# 若琪联盟 APP 介绍

## 什么是 若琪联盟 APP

若琪联盟 APP 直接可以对 搭载 Rokid服务的设备配网、指令发送、技能控制、ioT、设备管理等交互能力。 

目前支持手机系统有：iOS（苹果）、Android（安卓）。
